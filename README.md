# Vehicle Registration System (C++)

A console-based application developed in C++ to manage vehicle records efficiently.  
This project allows users to register vehicles, search for specific records, and display all stored data.

---

## Features
- Register new vehicles with details
- Search vehicle using registration number
- Display all registered vehicles
- Menu-driven user interface

---

## 🛠️ Tech Stack
- C++
- Standard Library (iostream, string)

---

## How to Run

1. Compile the code:
   g++ vehicle_registration_system.cpp -o vehicle

2. Run the program:
   ./vehicle

3. Follow the menu instructions
